# DSA Visualizer — VS Code Extension

Adds a "Visualize Current File" command (`Ctrl/Cmd+Alt+V`) that:

1. Reads the active editor's contents.
2. POSTs to the configured backend (default `http://localhost:8000`).
3. Opens a webview panel rendering the trace using `visualizer-core`.

## Settings

| Key                  | Default                  | Notes                                   |
| -------------------- | ------------------------ | --------------------------------------- |
| `dsaViz.backendUrl`  | `http://localhost:8000`  | Where to send `/trace` requests.        |
| `dsaViz.maxEvents`   | `5000`                   | Trace size cap.                         |

## Build

```bash
npm install
npm run compile
code --extensionDevelopmentPath=.
```

Then run `DSA Visualizer: Visualize Current File` from the command palette.

## How the webview connects to `visualizer-core`

The webview's HTML is generated by `extension.ts`. It bundles
`@dsa-viz/visualizer-core` via `esbuild` or `vite` (whichever the host repo
prefers) and uses the same `<VisualizerPanel/>` component as the browser
extension and standalone web app.
